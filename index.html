Step 1 — Auto-Sync Setup (Supabase — Free)

This gives you a real database. Add a transaction on your laptop — it instantly appears on your phone, tablet, anywhere. Free forever for your usage level.

1
Create a free Supabase account
Go to supabase.com → Sign up → New project. Choose any name (e.g. "portfolio") and any region. Wait ~1 min for it to spin up.
2
Create the database table
In Supabase: go to SQL Editor → paste and run this SQL:
create table portfolio (id text primary key, data jsonb not null, updated_at timestamptz default now());
3
Get your Project URL and Anon Key
Go to Project Settings → API. Copy:
• Project URL (looks like: https://xxxx.supabase.co)
• anon public key (long string starting with "eyJ…")
4
Paste them into the purple bar above
Paste the Project URL and Anon Key into the ☁ Auto-Sync bar at the top → click Connect. Your data syncs automatically from that point.
5
Host the HTML file (optional but recommended)
Upload this HTML file to GitHub Pages or Cloudflare Pages (both free). Then open the URL on any device — your Supabase credentials are saved and your data is always in sync.
Step 2 — Host on GitHub Pages (Free)
1
Create a GitHub repo
github.com → New repository → name it "portfolio" → Public → Create.
2
Upload this file as index.html
Click "uploading an existing file" → drag this file → rename to index.html → Commit.
3
Enable GitHub Pages
Settings → Pages → Source → Deploy from branch → main / root → Save. Live in ~60 seconds at:
https://YOUR-USERNAME.github.io/portfolio/
